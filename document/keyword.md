# Type inference
Type inference là khả năng Go tự suy luận kiểu dữ liệu của biến từ giá trị được gán, giúp bạn không cần khai báo kiểu một cách tường minh.
# Type conversion
Type conversion là khả năng Go chuyển đổi kiểu dữ liệu của biến từ một kiểu sang một kiểu khác, giúp bạn có thể sử dụng các hàm và toán tử với các biến có kiểu khác nhau.
# Type assertion
Type assertion là khả năng Go kiểm tra kiểu dữ liệu của biến và có thể sử dụng nó nếu nó có kiểu đúng.
